# 💹 Stock Portfolio Tracker

A beginner-friendly Python command-line app that lets you build a stock portfolio,
calculate total investment value, and export results to TXT or CSV.

Built as a practical exercise covering core Python concepts:
dictionaries, user input, arithmetic, and file handling.

## Features

- 10 pre-loaded stocks with hardcoded prices (AAPL, TSLA, NVDA, and more)
- Add multiple stocks with custom quantities
- Live value calculation per holding with allocation percentage
- Formatted portfolio summary table in the terminal
- Export report as `.txt` or `.csv` (or both)

## Concepts Covered

| Concept | Usage |
|---|---|
| Dictionary | Stock price lookup, portfolio storage |
| Input / Output | User prompts, terminal display |
| Arithmetic | Price × quantity, totals, percentages |
| File Handling | Writing `.txt` reports and `.csv` exports |

## How to Run

```bash
python stock_portfolio_tracker.py
```
No external libraries required — uses Python standard library only.
